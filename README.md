🚀 Rocket Boost – 3D Unity Game

Rocket Boost is a simple 3D physics-based rocket control game built using Unity and C#.

In this game, the player controls a rocket and must safely travel from the launch pad to the landing pad while avoiding obstacles. Precision, timing, and control are important to successfully complete each level.

The game currently contains 3 levels. When you complete Level 1 you move to Level 2, then Level 3, and after completing Level 3 the game loops back to Level 1.

🎮 Controls
Key	Action
Space Bar	Apply rocket thrust
Left Arrow	Rotate rocket left
Right Arrow	Rotate rocket right

While playing, you must carefully control the rocket's movement to avoid crashing into obstacles.

⚠️ Collision System

If the rocket hits an obstacle, the level will restart.

The restart happens after 3 seconds.

Sound effects play when:

The rocket collides with an obstacle

The rocket uses thrust

# 🚀 Rocket Boost – 3D Unity Game

Rocket Boost is a simple **3D rocket control game** built using **Unity** and **C#**.  
The goal of the game is to control a rocket and safely travel from the **Launch Pad** to the **Landing Pad** while avoiding obstacles.

The game currently contains **3 levels**.  
After completing **Level 1 → Level 2 → Level 3**, the game loops back to **Level 1**.

---

# 🎮 Controls

| Key | Action |
|-----|-------|
| **Space Bar** | Apply rocket thrust |
| **Left Arrow** | Rotate rocket left |
| **Right Arrow** | Rotate rocket right |

Use careful control and timing to avoid obstacles and reach the landing platform.

---

# ⚠️ Game Mechanics

- The rocket must travel from **Launch Pad → Landing Pad**
- Touching an **obstacle will crash the rocket**
- The level will **restart after 3 seconds**
- Sound effects play when:
  - Rocket **collides with an obstacle**
  - Rocket **uses thrust**
